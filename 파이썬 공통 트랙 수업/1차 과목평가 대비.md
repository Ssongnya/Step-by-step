#### Q. 다음 코드의 결과로 올바른 것을 고르시오. 
```python
x = 0

print(x or 5)
```
내가 고른 오답 : True

정답 : 5

#### 해설
> or 연산자는 앞의 값이 False로 평가되면 뒤의 값을 반환한다. 0은 False로 간주되므로 5가 출력된다.

#### Q. 다음 코드 실행 시 출력 결과로 옳은 것은?


내가 고른 오답 : a
